# 🧊 Stickstofftank Monitor v7

Ein Web-basiertes Dashboard zur Überwachung von Stickstofftank-Füllständen mit automatischer Verlustrate-Berechnung und Live-Prognose.

## Features

- 📊 Echtzeit-Tankstand mit Prognose
- 📈 Verlaufsanalyse mit Verbrauchsrate
- ⚠️ Anpassbare Warnschwellen
- 💾 CSV-Datenverwaltung
- 🔄 Import/Export Funktion
- 📱 Mobile-optimiert

## Installation

```bash
pip install -r requirements.txt
streamlit run claude_tank_monitor_v7.py
```

## Verwendung

1. Messwert eingeben
2. "Verlust" oder "Nachgefüllt" klicken
3. Dashboard zeigt aktuelle Prognose
